# Laptop Store E-commerce Platform

A monolithic e-commerce web application for selling laptops, built using Java and Spring Boot.

## Features
- **Admin Management System**: Complete dashboard and management tools.
- **Product Management**: CRUD operations for products, categories, and inventory.
- **Order Processing**: End-to-end checkout and order management system.
- **Security**: Robust authentication and authorization using Spring Security and JWT.
- **Statistics Dashboard**: Sales and product performance tracking.

## Technology Stack
- **Backend**: Java, Spring Boot, Spring Security, Spring Data JPA
- **Database**: SQL Database
- **Authentication**: JSON Web Tokens (JWT)
